Pengetahuan

Web merupakan sebuah halaman yang didalamnya terdapat berbagai informasi untuk memudahkan kita berinteraksi dengan komputer. Pada pembuatan Web biasanya menggunakan HTML, CSS dan bahasa pemrograman JS.

HTML merupakan singkatan dari Hyper Text Markup Language yang biasanya digunakan untuk pembuatan kerangka web.

CSS merupakan singkatan dari Cascading Style Sheet yang biasanya digunakan untuk memperindah tampilan web.

JS merupakan singkatan dari JavaScript, biasanya JS digunakan untuk membuat web menjadi interktif dan dapat berinteraksi dengan pengguna.
